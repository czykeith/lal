# 快速修复指南

## 问题
```
fatal error: bcryptprimitives.dll not found
```

## 最快解决方法

### 方法 1：运行修复脚本（推荐）
1. 右键点击 `fix_windows_dll.bat`
2. 选择"以管理员身份运行"
3. 等待修复完成
4. 重启计算机
5. 重新运行程序

### 方法 2：手动运行命令
以管理员身份打开命令提示符（CMD），依次执行：

```cmd
sfc /scannow
```

等待完成后，再执行：

```cmd
DISM /Online /Cleanup-Image /RestoreHealth
```

然后再次执行：

```cmd
sfc /scannow
```

### 方法 3：检查 Windows 更新
1. 按 `Win + I` 打开设置
2. 进入"更新和安全" > "Windows 更新"
3. 点击"检查更新"
4. 安装所有可用更新
5. 重启计算机

## 如果仍然无法解决

请查看 `WINDOWS_DLL_FIX.md` 获取更多详细解决方案。

## 常见问题

**Q: 为什么会出现这个问题？**
A: 通常是 Windows 系统文件损坏或缺失导致的。

**Q: 修复需要多长时间？**
A: 通常 10-30 分钟，取决于系统状态。

**Q: 修复会删除我的文件吗？**
A: 不会，这些命令只修复系统文件，不会影响用户数据。

**Q: 修复后需要重启吗？**
A: 是的，建议重启以确保所有修复生效。
